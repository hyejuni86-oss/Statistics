# 1️⃣ 개념 정리 
## 01. 데이터를 분석하다

```
✅ 학습 목표 :
* 데이터에 대해 이해한다.
* 데이터 분석에 대해 이해한다.
```
대상을 관찰하고 측정함으로써, 그 대상의 정보 즉 '데이터'를 얻을 수 있다. 

데이터 분석의 목적

1. 데이터를 요약하는 것 

2. 대상을 설명하는 것 

3. 새로 얻을 데이터를 예측하는 것 


## 02. 통계학의 역할

```
✅ 학습 목표 :
* 산포에 대해 설명할 수 있다.
```

데이터 분석에서 통계학의 중요한 역할은, 퍼짐(산포)이 있는 데이터에 대해 설명이나 예측을 하는 것이다. 

퍼짐이란 데이터에 포함된 값 하나하나의 차이를 가리킨다. 

데이터 퍼짐은 대상이 가진 성질이나 관계성의 본모습을 감추고, 정확하게 파악할 수 없도록 한다. 

통계학에선 이런 데이터 퍼짐을 '불확실성' 이라 평가하고 통계학의 목적인 '대상의 설명과 예측' 을 수행한다. 


## 03. 통계학의 전체 모습
```
✅ 학습 목표 :
* 기술통계와 추론통계의 차이점을 설명할 수 있다.
* 통계적 추론과 가설 검정에 대해 이해한다.
```
기술통계: 수집한 데이터를 정리하고 요약하는 방법.

추론통계: 수집한 데이터로부터 데이터의 발생원을 추정하는 방법. 

*추론통계*

-통계적 추론: 데이터에서 가정한 확률 모형의 성질을 추정하는 방법.

-가설검정: 세운 가설과 얻은 데이터가 얼마나 들어맞는지를 평가하여, 가설을 채택할 것인가를 판단하는 방법. 

## 04. 데이터 분석의 목적과 알고자 하는 대상

```
✅ 학습 목표 :
* 데이터 분석의 목적에 맞게 ‘알고자 하는 대상(모집단)’을 명확히 정의하는 것이 왜 중요한지 이해한다.
```

구체적인 데이터 분석 목적을 정하는 것이 중요한 첫 단계이다. 

데이터 분석의 목적을 정했다면, 다음으로는 알고자 하는 대상을 명확히 하는 것이 중요하다. 




## 05. 모집단

```
✅ 학습 목표 :
* 모집단의 개념을 명확히 설명할 수 있다.
```

모집단: 알고자 하는 대상 전체. 

모집단 크기: 모집단에 포함된 요소.

유한모집단: 모집단 중 한정된 요소만 포함한 것. 

무한모집단: 모집단 중 포함된 요소의 개수가 무한한 것. 


## 06. 모집단의 성질을 알다 

```
✅ 학습 목표 :
* 전수조사와 표본조사의 개념적 차이와 각각의 필요성을 설명할 수 있다.
* 표본에 대해 이해한다.
```

모집단의 성질이란, 모집단에 포함된 요소를 특징 짓는 값이다. 

전수조사: 모집단의 성질을 아는 방법의 하나, 모집단에 포함된 모든 요소를 조사함. 

- 그러나 비용이나 시간 면에서 부담이 막대하여 실현 불가능할 때가 대부분임. 

표본: 추론통계에서 조사하는 모집단의 일부. 
표본추출: 모집단에서 표본을 뽑는 것. 
표본조사: 표본을 이용해 모집단의 성질을 조사하는 것. 

ex. 선거출구조사

표본크기: 표본에 포함된 요소의 개수

--> 표본크기는 모집단의 성질을 추정할 때의 확실성이나 가설검정의 결과에도 영향을 끼치기 때문에, 통계분석에 있어 중요한 요소 중 하나이다.

# 2️⃣ 확인 문제

## 문제 1.

> **🧚Q. 한 회사가 직원들의 ‘업무 만족도’를 파악하기 위해 설문조사를 실시했다.
전체 직원 1,000명 중 50명에게만 설문을 진행했고, 그 결과 평균 만족도는 4.5점(5점 만점)이었다.
회사는 이를 근거로 “우리 회사 직원들은 대체로 업무 만족도가 높다”고 결론 내리려 한다.**
>
> **이 결론이 타당한지 판단하기 위해 가장 먼저 확인해야 할 것은 무엇인가?**

~~~
1️⃣ 조사 대상 50명이 전체 직원을 대표하는 표본인가?
2️⃣ 만족도의 평균값만으로 결론을 내려도 되는가?
3️⃣ 설문 문항이 객관적으로 구성되었는가?
4️⃣ 이 결과를 모집단(전체 직원)으로 일반화해도 되는가?
~~~
```
가장 먼저 조사 대상 50명이 전체직원을 대표하는 표본인지 확인해야한다. 무작위로 추출이 되었는지, 사람들이 골고루 포함이 되었는지 확인해야한다. 표본의 대표성이 없으면 평균값 자체가 전체 직원 만족도를 제대로 반영하지 못하기 떄문이다. 
```
